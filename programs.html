Code


WeilRep is Sage code for calculations with vector-valued modular forms for Weil representations, Jacobi forms, theta lifts and Borcherds products.
Download it from GitHub. See the readme for an explanation of how to use it.
If you find a bug in this program or have other questions or comments please contact me via email.



Miscellaneous things:
  • Here is code for working with eta products. It computes the q-series of an eta product and tries to identify an eta product from a q-series.
  • Here is code for computing exact values of Dedekind zeta functions for totally real number fields.
  • Here is code that attempts to split hyperbolic planes off of an even lattice.
